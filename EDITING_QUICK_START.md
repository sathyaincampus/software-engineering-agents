## Common Editing Tasks

### Change Database from PostgreSQL to MongoDB

1. Click "Edit Architecture"
2. Find the database section:
```json
"database": {
  "primary": "PostgreSQL",
  "caching": "Redis"
}
```
3. Change to:
```json
"database": {
  "primary": "MongoDB",
  "caching": "Redis"
}
```
4. Click "Save Changes"

### Add a New User Story

1. Click "Edit User Stories"
2. Click "+ Add Story"
3. Fill in:
   - ID: US-010
   - Priority: High
   - Title: Password Reset
   - Description: As a user, I want to reset my password...
4. Click "Save Changes"

### Modify Mermaid Diagram

1. Click "Edit Architecture"
2. Find `"system_diagram"` â†’ `"code"`
3. Edit the Mermaid syntax:
```
"code": "graph TD\n  A[Client] --> B[API Gateway]\n  B --> C[Backend]"
```
4. Click "Save Changes"

1. **Your edits are saved** to the project files
2. **Code generation uses your edited content**
3. **Changes persist** when you reload the project
4. **Logs confirm** when edits are saved
